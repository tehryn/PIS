Manažer luxusního hotelu se rozhodl přejít z papírové evidence rezervací na elektronickou. Z tohoto důvodu si objednal nový informační systém, který recepci hotelu umožní rezervace spravovat.

Hotel kromě ubytování také nabízí různé služby klientů. Mezi tyto služby patří například pronájem bowlingové dráhy nebo kulečníkového stolu. Tyto služby si klient může rezervovat stejným způsobem jako pokoje.

Primární funkcí systému bude rezervace pokojů na konkrétní datum a počet nocí. Součástí rezervace bude výběr požadovaných služeb (například rezervace kulečníkového stolu). Systém umožní zobrazení dostupnosti pokojů a služeb neregistrovaným uživatelům. Registrovaný uživatel bude mít možnost prostřednictvím webového rozhraní požádat o rezervaci pokoje. Rezervaci bude moci potvrdit recepční, případně může vytvořit novou rezervaci na základě komunikace s klientem. Rezervaci bude mít možnost upravit jak autor rezervace, tak i recepční. Klient bude moci zobrazit historii u svých rezervací, recepční u všech rezervací. Manažer hotelu musí mít možnost zablokovat na určitá data pokoje i služby (např. kvůli rekonstrukci). Součástí systému bude i uživatel v roli administrátora, který bude mít veškerá oprávnění včetně správy uživatelů.

Postup instalace

Projekt je implementován v jazyce Java EE. Pro správnou funkcionalitu projektu je nutné před spuštěním nakonfigurovat připojení k databázovému serveru.

Databáze

Pro relační databázový model byl zvolen systém MySQL. Pro korektní nastavení databáze je třeba provést následující konfiguraci mysql serveru:

Editace konfiguračního souboru (na ubuntu se jedná o soubor /etc/mysql/my.cnf): 
    [mysqld]
    lower_case_table_names = 1
Vytvoření databáze: 
    CREATE DATABASE pis;
    ALTER DATABASE pis CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
    CREATE USER 'pis'@'localhost' IDENTIFIED BY 'p1sh3sl0';
    GRANT ALL PRIVILEGES ON pis.* TO 'pis'@'localhost' IDENTIFIED BY 'p1sh3sl0';
Vytvoření tabulek a naplnění testovacími daty: 
    mysql -u [USER] --password=[PASSWORD] -s <./sql/init.sql
